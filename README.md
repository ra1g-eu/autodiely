# Project overview (1.0.0)

## Running the project

### Laravel Backend
1. `cd backend`
2. `composer install`
3. `php artisan migrate`
4. `php artisan serve`

### Vue Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
4. `site accessible at http://localhost:3000`
